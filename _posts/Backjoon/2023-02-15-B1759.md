### [백준] 1759 암호만들기
체감 난이도 : 🎈🎈 **/5**   

백트래킹으로 구현한 암호만들기 문제.   

```python
def promising(i):
    # global word
    for k in range(i):
        if word[k] == word[i]:
            return False
    else:
        if i > 0 and ord(word[i-1]) > ord(word[i]):
            return False
        else:
            return True

def DFS(x):
    global word
    global res
    if x == L:
        if 1 <= len(set(word) & {'a','e','i','o','u'}) <= L-2:        
            res.append(''.join(word))
        else:
            return 
    else:
        for k in word_candidates:
            word[x] = k
            if promising(x):
                DFS(x+1)

L, C = map(int,input().split())
word_candidates = input().split()
word = ['']*L
res = []
DFS(0)

for _ in sorted(res):
    print(_)
```
